# Calculadora de Sub redes/Subnet Calculator
Portugês: Calculadora de sub-rede IPv4. Este script de Python faz o calculo de: Máscara de rede; Rede; 1º IP válido; Último IP válido; Broadcast; classe de IP; e tipo de IP. Além disso, ele indica qual erro você faz ao digitar, como: se não há somente números no IP e CIDR; quantidade de octetos; quais octetos não foram digitados; e se cada octeto e o CIDR está na faixa ideal.

English: IPv4 subnet calculator. This Python Script calculates: Subnet Mask; Network Address; 1º usable IP; last usable IP; Broadcast; IP class; and IP type. Furthermore, it indicates which mistake you made when you type, as: if there's not just numbers on IP and CIDR; numbers of octets; which octet weren't typed; and if each octet and CIDR is on the ideal range.
